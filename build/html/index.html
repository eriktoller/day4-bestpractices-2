simple_math.poly_first(x, a0, a1)¶

This fucntion resturns the first order polynomial y = a0 + a1*x.

Parameters
a0: float, a constant
a1: float, the multiplicand
x: float, the multiplier
Returns
a0 + a1*x: float, result of the first order polynomial
>>> poly_first(2,5,3)       
    11
simple_math.poly_second(x, a0, a1, a2)¶

This fucntion resturns the second order polynomial y = a0 + a1*x + a2**x. The function calls the poly_first(x, a0, a1) funciton.

Parameters
a0: float, a constant
a1: float, the multiplicand
a2: float, the base
x: float, the multiplier and exponent
Returns
a0 + a1*x + a2**x: float, result of the second order polynomial
>>> poly_first(2,5,3,6)     
    35
